# ExamUp iOS

Приложение для подготовки к ЕГЭ по математике — профильный уровень. Интерактивные задачи, подсказки, разбор ошибок и отслеживание прогресса через Firebase.

[![App Store](https://img.shields.io/badge/App%20Store-Download-blue?logo=apple)](https://apps.apple.com/app/examup/id6744438062)

## ✨ Возможности

- Задачи профильного ЕГЭ по всем темам
- Пошаговые подсказки и разбор решений
- История попыток и статистика по темам
- Синхронизация через Firebase Firestore
- Адаптивный SwiftUI-интерфейс

## 🛠 Стек

- **Swift** + **SwiftUI**
- **Firebase** (Firestore, Analytics, Crashlytics)
- **Combine** — реактивные обновления
- Архитектура: MVVM + Clean Architecture

## 🚀 Сборка

```bash
git clone https://github.com/FeofanovIvan/examup-ios.git
cd examup-ios
open ExamUp.xcodeproj
```

Требования: Xcode 15+, iOS 16+

> ⚠️ `GoogleService-Info.plist` исключён из репозитория. Добавь свой файл из Firebase Console.
